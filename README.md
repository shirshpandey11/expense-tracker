# Expense Tracker

## Overview

Expense Tracker is a full-stack web application that helps users manage their daily expenses. Users can add, edit, delete, and filter expenses while also viewing spending insights through summary cards and charts.

## Features

* Add new expenses
* Edit existing expenses
* Delete expenses
* Filter by category
* Filter by date range
* View total spending
* View highest expense
* View total number of expenses
* Visualize expenses using charts

## Project Structure

The project consists of two main parts:

**Client (Frontend)** – Built using React and Vite. It contains the user interface, forms, filters, summary cards, and charts.

**Server (Backend)** – Built using Express.js. It handles API requests and manages expense data stored in a JSON file.

---

## How It Works

1. Users enter expense details through the form.
2. The React frontend sends requests to the Express backend.
3. The backend stores expense data in a JSON file.
4. Expenses are displayed on the dashboard.
5. Summary cards and charts update automatically.
6. Users can edit, delete, and filter expenses.

## Setup Instructions

### Clone the Repository

```bash
git clone https://github.com/shirshpandey11/expense-tracker.git
cd expense-tracker
```

### Backend Setup

```bash
cd server
npm install
node server.js
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```
